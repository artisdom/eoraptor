# eoraptor
Simple tool to scan for wasteful PNG text chunks in PE files.

Keep in mind, the tool wasn't designed to measure exactly (i.e. it doesn't count infrastructure needed to host text chunks).

# contributing
Compiled with Visual Studio 2017 on Windows 10 Creators Update.

Pull requests welcome!
